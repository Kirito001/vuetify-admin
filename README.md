# vuetify-admin
一个vuetify后台基础模板
## 预览地址
<a href="http://fhluo.gitee.io/vuetify-admin" target="_blank">在线预览</a>
## 使用步骤
##### 1.克隆项目到本地
`git clone https://github.com/Kirito001/vuetify-admin.git`
##### 2.进入刚下载的项目目录
`cd vuetify-admin`
##### 3.安装依赖
`npm install`
##### 4.运行项目
`npm run serve`
## 打包成静态文件
`npm run build`
